# 텍스트 서식 변환

마크다운/HTML 인라인 서식을 MediaWiki 문법(또는 허용 HTML 태그)으로 변환합니다.

| 서식            | MediaWiki 표현                              |
| --------------- | ------------------------------------------- |
| 코드 블록       | `<syntaxhighlight lang="{언어}"> ... </syntaxhighlight>` |
| 볼드            | `<b> ... </b>`                              |
| 강조            | `<strong> ... </strong>`                    |
| 이탤릭          | `<i> ... </i>`                              |
| 인라인 코드     | `<code> ... </code>`                        |
| 취소선          | `<del> ... </del>`                          |
| 밑줄            | `<u> ... </u>`                              |
| 위첨자          | `<sup> ... </sup>`                          |
| 아래첨자        | `<sub> ... </sub>`                          |
| 작은 글씨       | `<small> ... </small>`                      |
| 큰 글씨         | `<big> ... </big>`                          |
| 인용 블록       | `<blockquote> ... </blockquote>`            |
| 줄바꿈          | `<br />`                                    |
| 사전 서식(언어 미지정) | `<pre> ... </pre>`                    |
| 수평선          | `<hr />`                                    |
| LaTeX 수식      | `<math> ... </math>`                        |
| 문법 무시       | `<nowiki> ... </nowiki>`                    |
