# Unordered List 변환

GeekNews의 마크다운 불릿 리스트를 MediaWiki 문법으로 변환합니다.

| 마크다운        | MediaWiki         |
| --------------- | ----------------- |
| `- 항목` / `* 항목` | `* 항목`          |
| 2단계 중첩      | `** 항목`         |
| 3단계 중첩      | `*** 항목`        |

본문 상단의 Unordered List(첫 번째 수평선 위에 위치한 불릿 리스트)는 **편집 없이 원문 그대로** MediaWiki 문법만 치환하여 가져옵니다.
