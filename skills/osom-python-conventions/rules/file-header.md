# 파일 헤더

모든 `.py` 파일은 UTF-8 인코딩 선언으로 시작합니다.

```python
# -*- coding: utf-8 -*-
```

## 적용 기준

- 모든 새 `.py` 파일 생성 시 첫 줄에 반드시 포함
- 기존 파일 수정 시 누락되어 있으면 추가
- 빈 `__init__.py`에도 인코딩 선언만 두는 것을 권장 (선택 사항)

## 예외

- shebang이 필요한 실행 스크립트는 shebang을 먼저 둡니다.

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
```

## 이유

- Python 3.x는 기본 UTF-8을 사용하지만, 명시적 선언은 에디터·도구 호환성과 의도 표명에 유리합니다.
- 일관된 헤더는 grep·sed 등으로 파일 메타데이터를 다룰 때 편리합니다.
