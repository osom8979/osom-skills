# 페이지 전용 하위 컴포넌트

페이지가 복잡해지면 **언더스코어 프리픽스 파일**(예: `_FeatureCard.tsx`)로 같은 디렉토리에 분리합니다.

이 컴포넌트는:

- **해당 페이지에서만 사용됨** — 재사용 금지
- **Controlled Component로 구현** — props로 데이터 수신
- 가능하면 스토리와 테스트를 함께 생성

페이지 외부에서 import하지 않도록 언더스코어 prefix가 시각적 신호를 줍니다. 만약 다른 페이지에서도 쓰게 되면 즉시 `<components-root>/`로 승격하고 prefix를 제거합니다.
