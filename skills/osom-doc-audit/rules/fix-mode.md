# 수정(fix) 모드

사용자가 `/doc-audit fix`로 호출하면, 검출된 불일치를 자동 수정합니다.

## 자동 수정 대상

- 문서에 누락된 파일/항목 추가
- 문서에 남아 있는 삭제된 파일/항목 제거
- 깨진 인덱스 링크 수정 (대상이 명확한 경우)

## 수정하지 않는 것

- **코드 스타일 위반은 자동 수정하지 않습니다** — 리팩토링 범위이므로 `osom-refactor`로 위임.
- 파괴적 변경(문서 대량 삭제, 인덱스 전면 재작성 등)은 사용자 확인을 받습니다.
- 의미가 불분명한 항목(어느 섹션에 추가할지 모호한 경우)은 보고만 하고 사용자 결정을 요청합니다.
