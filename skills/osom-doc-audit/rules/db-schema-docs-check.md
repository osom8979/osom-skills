# DB 스키마 문서 검증

프로젝트가 `postgres/` 같은 DB 스키마 디렉토리를 가지면, 관련 규칙 문서(`structure.md`, `conventions.md`, `sql-naming.md` 등)가 실제 디렉토리 구조와 일치하는지 확인합니다.

확인 대상:

- 도메인 디렉토리(번호 밴드) 일관성
- SQL 파일 네이밍 밴드(예: `10-tables-*`, `40-policies-*`) 일치 여부
- 문서에 나열된 테이블/함수/정책이 실제 파일에 존재하는지

프로젝트가 DB 레이어가 없으면 이 항목은 자동 제외됩니다.
